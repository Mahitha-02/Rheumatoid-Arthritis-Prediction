Rheumatoid Arthritis Prediction

Overview
This project predicts the likelihood of Rheumatoid Arthritis using Machine Learning algorithms.

Algorithms Used
- Logistic Regression
- Random Forest
- Decision Tree
- KNN
- SVM
- Naive Bayes

 Technologies
- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit

Features
- Disease prediction
- Model comparison
- Visualization charts
- User-friendly interface
